# Project-specific agents

This folder holds custom agents scoped to the Teachers Onboarding project.

## Rules

- **New agents require Yuval's explicit approval.** Do not create one without asking.
- Only add an agent when Dasha's general agents (`architect`, `implementer`, `researcher`, `test-writer`, `code-reviewer`, etc.) are genuinely insufficient for the task.
- Keep agents narrow and purposeful. If it looks like a general-purpose agent, it belongs in `~/.claude/agents/`, not here.
